# Windows-10-Setup-Personal
Scripts to setup Windows 10 PCs.


How to use:

    1 - Install Windows and Setup Account
    2 - Login
    3 - Open Edge and download this repo
    4 - Run PS as Admin
    5 - Execute 
        set-executionpolicy -ExecutionPolicy Bypass -Scope Process -Force
    6 - Run the script
        ./setup.ps1
